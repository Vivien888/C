答：
（1）描述：定义了一个print 函数，函数的参数等同于一个常量整型指针 const int*，形参 ia 的维度10，打印出数组的每一个元素。

（2）问题：上述 print 函数的定义存在一个潜在风险，
                   即虽然我们期望传入的数组维度是10，但实际上任意维度的数组都可以传入。
                   如果传入的数组维度较大，print 函数输出数组的前10个元素，不至于引发错误；
                   相反如果传入的数组维度不足10，则 print 函数将强行输出一些未定义的值。

（3）修改如下：
void print(const int ia[], const int sz)
{
	for (size_t i = 0; i != sz; ++i)
		cout << ia[i] << endl;
}
